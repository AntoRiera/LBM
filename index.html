Cada mesa cuenta una historia diferente

Mesa para dos

Una mesa pensada para compartir de cerca. Ideal para una cita, una sorpresa romántica o simplemente un momento entre dos, cuidado hasta el último detalle.

Un picnic íntimo, tranquilo y especial, donde todo invita a disfrutar sin prisa: la comida, la conversación, el entorno y el tiempo juntos.

Mesa compartida

Una mesa para reunirse con amigos y dejar que el plan fluya. Pensada para brindar, conversar, reírse y quedarse un poco más, con una estética cuidada pero relajada.

Es la opción ideal para quienes quieren disfrutar de un encuentro al aire libre sin complicarse, pero con todos los detalles preparados.

Mesa de celebración

Una mesa creada para momentos que merecen algo especial. Cumpleaños, sorpresas, despedidas, aniversarios o cualquier ocasión que quieras recordar de una forma distinta.

Diseñamos la experiencia alrededor del motivo de celebración, cuidando la estética, los detalles y el ambiente para que el momento se sienta único desde el principio.
