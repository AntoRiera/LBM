# La Buena Mesa — web estática

Proyecto HTML/CSS/JS listo para publicar en GitHub Pages.

## Archivos principales

- `index.html`: estructura y textos de la página.
- `styles.css`: estilos desktop y mobile.
- `script.js`: menú mobile y header con scroll.
- `assets/`: imágenes, logo e iconos.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube `index.html`, `styles.css`, `script.js` y la carpeta `assets`.
3. Entra en `Settings` → `Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama `main` y carpeta `/root`.
6. Guarda y espera unos minutos.

## Hacer cambios

- Textos: editar `index.html`.
- Colores, tamaños, responsive/mobile: editar `styles.css`.
- Imágenes: reemplazar archivos dentro de `assets` manteniendo el mismo nombre.
